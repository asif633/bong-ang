How to use the app?

1. Sign in using your google account, using left side bar

2. Choose Angular2 Firebase app in the side bar

3. Start by naming your project and choosing the css framework

4. Change theme colors if available and needed

5. Next tab add your models along with their specific fields

6. Choose available widgets and map to the specific fields

7. Click on generate app

8. App download link will appear

9. Extract the app, and cd to app folder

10. You will need angular-cli to run the app

11. You should have global angular cli; npm install -g @angular/cli

12. Then run npm install

13. Lastly run ng serve
